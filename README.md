Я Антон, Data Engineer. Кодю на Python, запросю на SQL.

### 🔍 Что здесь у нас? Скрипты на Python и утилиты автоматизации

Личный полигон для проверки кода, тестирования библиотек (requests, pandas) и обкатки различных решений.

### 📂 Структура репозитория

* **`Greenplum to Kafka.py`** — Скрипт-прототип для стриминга данных из базы Greenplum в очереди Kafka.
* **`city_weather.py` & `currency_api.py`** — Парсеры для работы с внешними REST API. Собирают данные о погоде и курсах валют (формат JSON), обрабатывают ошибки сети и пишут результат в CSV.
* **`telegram_errors_bot.py`** — Бот для отправки системных уведомлений и логов о сбоях пайплайнов в Telegram.
* **`json_yaml_csv_rw.py`** — Скрипт для конвертации данных между форматами JSON, YAML и CSV.
* **`simple_calculator.py` & `even_numbers.py`** — Простые скрипты для проверки базового синтаксиса Python.
* **`config/`** — Папка с общим конфигурационным файлом `global_config.py` для хранения токенов и путей. ⚙️
* **`requirements.txt`** — Список требуемых библиотек для этиъ скриптов.

---

### 🚀 Запуск

Для установки всех внешних зависимостей (requests, pyyaml, pandas) выполнить в терминале:

pip install -r requirements.txt
